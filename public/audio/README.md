# Calibration Audio Files

This directory contains public domain audio files for calibration and testing.

## Available Songs

Add MP3 files here for calibration. Good sources for public domain music:

1. **Musopen** - https://musopen.org/music/ (Classical music)
2. **FreePD** - https://freepd.com/ (Various genres)
3. **Public Domain 4U** - http://www.pdmusic.org/
4. **Internet Archive** - https://archive.org/details/audio

## Recommended Test Files

For calibration, simple songs with clear notes work best:
- Scale exercises (C major scale)
- Simple melodies (Twinkle Twinkle, Mary Had a Little Lamb)
- Single instrument recordings
- Known BPM tracks for beat detection testing

## File Format

- Format: MP3 or WAV
- Naming: Use descriptive names (e.g., "c-major-scale.mp3", "120bpm-test.mp3")
