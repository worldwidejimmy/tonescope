 ToneScope Calibration Tracks – README.txt
===========================================

Overview
--------
This calibration pack contains public-domain (CC0) music recordings from FreePD.com.
They are ideal for testing real-time audio analysis features in ToneScope:
 • Key / scale detection
 • Note tracking
 • Beat and tempo estimation
 • Amplitude and loudness visualization

All files are 100% safe and royalty-free for commercial and personal use.

Directory Structure
-------------------
full/   – Complete tracks (1–3 minutes each)
clips/  – Short ~25 second calibration excerpts
LICENSE.txt – Confirms CC0 / Public Domain status of all included works
README.txt  – This document
